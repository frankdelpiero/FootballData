# FootballData
A Sports Analytics project transforming raw football data into actionable match insights. Using Power BI, I analyze tactical behaviors, offensive efficiency, and disciplinary risks to predict match outcomes and player performance.
## 📝 Overview
This project transforms raw LaLiga performance data into strategic insights. Using **Power BI**, I analyze tactical behaviors, offensive efficiency, and disciplinary risks to predict match outcomes.

## 🚀 Live Insights (Case Studies)
* **Real Betis vs. Rayo Vallecano:** Identified the "Friction Zone" caused by Rayo's high foul volume (Lejeune/Chavarría) vs. Betis' electric wingers.
* **Real Oviedo's Wall:** Analyzed how a 75% Save% creates a strong "Under 2.5/No BTTS" trend despite low offensive output.

## 🔍 Key Analytical Focus
* **Offensive Efficiency:** Comparing **Gls/90** vs. **G/SoT** to evaluate clinical finishing.
* **Disciplinary Risk:** Correlation between **Fouls** and **Yellow Cards** to identify high-risk players.
* **Set Piece Projection:** Using **Crosses** and **Shot Volume** to project Corner Kick outcomes.

## 🛠️ Tech Stack
* **Data Source:** FBRef / StatsBomb data.
* **Tool:** Power BI (Advanced DAX & Power Query).
* **Visualizations:** Scatter plots for risk analysis, Gauge charts for efficiency, and KPI cards for player impact.
